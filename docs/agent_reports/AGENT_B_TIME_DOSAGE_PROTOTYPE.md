# AGENT_B_TIME_DOSAGE_PROTOTYPE

## Summary

完成时间投入原型：符合条件的训练/工作/观察行动会先选择 30/60/90/120 分钟投入，再按 multiplier 结算收益、体力、疲劳和硬练风险。

## Gameplay Notes

- 标准 60m 保持默认收益倍率。
- 短练降低收益和疲劳，支持更多碎片行动。
- 深练提高收益但显著加疲劳。
- 硬练收益递减，附加疲劳和小伤概率，不应成为无脑最优。
- 发呆/放空只给少量恢复与叙事触发，不产出现金、技能大量 XP 或训练核心收益。

## UI Notes

- 行动卡显示“可调时长”标记。
- 点击行动后弹出小选择框，列出耗时、体力、现金、倍率和风险。
- 训练小游戏会继承已选投入档位。
- 技能来源详情默认展开，保留 playtest 对来源可见性的断言。
- 地图推荐栏非交互区域不再挡住主线按钮点击。

## Validation

- `npm run check:full`：通过。
- `npm run test:playtest`：通过。
- `git diff --check`：通过。

## Risk

倍率和硬练风险为原型默认值；后续应根据真实游玩数据微调 `TIME_DOSAGE_OPTIONS`。
