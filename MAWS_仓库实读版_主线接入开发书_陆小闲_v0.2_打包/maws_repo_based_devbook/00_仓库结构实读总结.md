# 仓库结构实读总结

## 入口
- `maws_30day_overhaul_v3.html` 加载 `maws_src/dom/ui.css`、`vendor/phaser-4.1.0.min.js`、`maws_src/main.js`。
- 页面根节点为 `#game-root` 和 `#maws-ui-root`。

## 启动链路
- `maws_src/main.js` 创建 `GameStore`，初始化 DOM UI，启动 Phaser `BootScene` 与 `ShellScene`。

## 核心源码
- `maws_src/content/data.js`：静态数据、地点、行动、技能、物品、敌人、NPC、主线。
- `maws_src/simulation/state.js`：存档状态、时间、出行、训练、战斗生命周期、render model。
- `maws_src/simulation/events.js`：机会卡与权重。
- `maws_src/simulation/combat.js`：战斗公式、AI、自动窗口。
- `maws_src/dom/ui.js` / `ui.css`：DOM UI、地图、人物、技能、背包、商店、关系、日志、自检、战斗 HUD、弹窗。
- `maws_src/assets/manifest.js`：背景、角色、NPC、sprites、头像、物品、图标、技能卡、UI、VFX。
- `maws_src/tools/verify_assets.mjs`：资源校验。
