## 4. 茂家拳谱 UI

建议插入到 `renderProfile()` 的人物信息后、基础属性前。

### 卡片结构

```html
<section class="maws-punchbook">
  <header>
    <b>茂家拳谱</b>
    <span>完成度 42%</span>
  </header>
  <article class="maws-form-card locked|revealed|reforged">
    <strong>茂家开山拳</strong>
    <p class="old-note">旧解：一拳开山，气吞山河。</p>
    <p class="new-note">新注：先别开山，先把手收回来护脸。</p>
    <meter value="42" max="100"></meter>
  </article>
</section>
```

### 状态

- `locked`：Day 9 前只显示旧解。
- `revealed`：Day 9 后显示“旧解存疑”。
- `reforged`：重铸进度达标后显示红批新注。

## 5. Day 8 一阵风弹窗

```text
一阵风

你看见他的肩膀动了一下。
然后世界安静了。

爸，我只感觉到了一阵风。
```

按钮文案：

```text
醒过来
嘴硬一下
```

如果玩家点“嘴硬一下”：

```text
你试图解释这是茂家拳的试探性昏迷。
没有人接话。
```

## 6. Day 9 父亲日记

```text
父亲日记

儿子，如果你看到这里，大概是终于遇到真正会打的人了。

爸不是不教你绝招。
爸是真的不会。

茂家拳不是天下第一。
当年那个所谓祖师，是个快饿死的外乡人。
你太爷爷给了他几顿饭，他教了几手拳脚。
后来越传越神，传到我这儿，已经快能打雷了。

但你小时候总低着头。
别人笑你胖，笑你跑两步就喘。
爸不知道怎么劝你，只好告诉你：
我们家有天下第一的拳。

你信了。
你也真的开始练了。

如果真的遇到高手，退。
不是丢人，是回家。
```

最后按钮：

```text
原来都是假的
至少他是真的希望我站起来
从我开始，把它练成真的
```

## 7. Day 29 烧烤夜谈

可选对象：

- 梁教练
- 小满
- 刘胖子
- 周青山
- 阿豹
- 一个人

### 梁教练

```text
梁教练把烤串翻了个面。

“明天别想着证明传统武术。”
“你先证明自己这二十多天没白练。”
“护脸。别闭眼。挨打了也别把脑子一起丢出去。”
```

获得：

```text
状态：别想太多，护脸
效果：终战第一次重击时冷静检定 +8
```

### 小满

```text
小满递给你一瓶水。

“你明天要是输了，也别急着把自己说成骗子。”
“人不是因为赢了才算真的。”
“你爸想让你站起来，不是想让你被打到站不起来。”
```

获得：

```text
状态：有人等你回来
效果：终战士气崩溃阈值降低
```

### 一个人

```text
你点了十串羊肉。
其中九串献给了焦虑。
一串献给了父亲。

你忽然想起小时候第一次跑完一圈。
你爸说，茂家拳第一步，先别停。
```

获得：

```text
父亲记忆 +8
```

## 8. Day 30 结尾文案

### 失败但有成长

```text
你还是输了。
但这一次，你记得每一拳是怎么来的。
这已经不是一阵风了。
```

```text
你没有证明茂家拳天下第一。
但你证明了，它终于不是零。
```

```text
陈见锋看了你一会儿。

“你这拳乱七八糟。”

茂家拳谱第一页新注：
本门拳法，祖传部分存疑。
但今日起，有人开始认真练。
```

## 10. 旁白风格模板

### 误判胜利

```text
你摆出茂家拳起手式。
从远处看，确实有一代宗师的气势。
从近处看，像一个准备搬冰箱的人。
```

### 训练失败

```text
你的上半身同意了这个动作。
你的脚没有收到通知。
```

### 真实进步

```text
这一次，你看见了拳。
虽然还是慢了一点，但至少不是等世界变白以后才知道发生了什么。
```
