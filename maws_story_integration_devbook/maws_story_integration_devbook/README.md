# 《了不起的武术模拟器》主线接入开发书包

本包用于 `chasexu168340-rgb/maws-30day-overhaul` 当前 Phaser + DOM Web demo。

文件说明：

- `00_DEVBOOK.md`：总开发书，说明主线如何接入现有项目。
- `01_DATA_MODEL_PATCH_SPEC.md`：数据结构、状态字段、主线事件、NPC、装备、技能接入规格。
- `02_UI_AND_COPYDECK.md`：UI 展现方案、关键弹窗、拳谱页、文案风格和示例文案。
- `03_COMBAT_AND_NUMBERS.md`：战斗目标、数值规则、Day 8 “一阵风”、Day 30 目标制终战。
- `04_IMPLEMENTATION_TICKETS.md`：按 Batch 拆分的开发任务、涉及文件和验收标准。
- `05_QA_CHECKLIST.md`：功能、剧情、战斗、移动端、存档迁移测试清单。
- `maws_story_patch_snippets.js`：可参考的数据补丁片段，不建议整段直接覆盖现有文件。

注意：本包是开发书和落地规格，不是已提交到 GitHub 的代码补丁。
